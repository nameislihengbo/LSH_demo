## 一、引用格式标准 / Citation Format Standard

### 1.1 外部引用（已发表论文）

采用 `plainnat` 格式，遵循国际学术标准：

```
\bibitem[Author(et~al.)Year]{cite_key}
First~Author, Second~Author, and Third~Author.
\newblock Title of the paper in sentence case.
\newblock In \emph{Conference/Journal Name}, Year.
```

**字段要求**：

| 字段 | 必需 | 说明 | 示例 |
|------|------|------|------|
| Author | ✅ | 完整姓名，非缩写 | `Ashish Vaswani` |
| Title | ✅ | 句首大写，专有名词保留 | `Attention is all you need` |
| Venue | ✅ | 会议/期刊全名 | `NeurIPS`, `ICML`, `ICLR` |
| Year | ✅ | 四位数字 | `2017` |
| Pages | 推荐 | 起止页码 | `5998--6008` |
| Volume | 推荐 | 期刊卷号 | `378` |
| DOI | 可选 | 数字对象标识符 | `10.xxxx/xxxxx` |

### 1.2 内部引用（LSH 协议论文）

```
\bibitem[Li(2026a)]{cite_key}
Hengbo Li.
\newblock Full Title of the Paper.
\newblock \emph{LSH Protocol Preprint}, 2026.
```

**内部引用键命名规范**：

| 格式 | 说明 | 示例 |
|------|------|------|
| `lsh-sfa` | 核心架构论文（含时空引擎） | LSH-SFA + Spacetime Engine |
| `lsh-three-layer` | 三层架构论文（含执行层） | Three-Layer + Execution Layer |
| `encoding-gradient` | 编码与梯度论文（含梯度范式） | Tokenizer-Free + Controllable Explosion |
| `lsh-format` | LSH 格式论文（含 Rust 优化） | LSH Format + Rust/Burn Engineering |
| `attention-comparison` | 注意力对比论文 | Beyond Softmax |
| `autoregressive` | 自回归生成论文 | Autoregressive Generation |
| `lsh-burn` | LSH-Burn 系统论文 | Rust Implementation |
| `lsh-rules` | LSH 规则系统论文 | Self-Verifying System |
| `version-control` | 版本控制论文 | Ternary Semantic Versioning |
| `ethics-manifesto` | 伦理宣言论文 | Ethics Manifesto |

**已合并的引用键（不再使用）**：

| 旧引用键 | 合并到 | 说明 |
|---------|--------|------|
| `spacetime-engine` | `lsh-sfa` | 时空引擎已合并入 LSH-SFA 主论文 |
| `gradient-paradigm` | `encoding-gradient` | 梯度范式已合并入编码与梯度论文 |
| `rust-optimization` | `lsh-format` | Rust 优化已合并入 LSH 格式论文 |
| `lsh-execution-layer` | `lsh-three-layer` | 执行层已合并入三层架构论文 |

### 1.3 软件引用

```
\bibitem[Project(Year)]{cite_key}
Project Contributors.
\newblock Project Name: Brief Description.
\newblock \url{https://github.com/org/repo}, Year.
```

## 三、作者姓名规范 / Author Name Format

### 3.1 英文作者

| 规则 | 正确 | 错误 |
|------|------|------|
| 完整名+姓 | `Ashish Vaswani` | `A. Vaswani` |
| 连字符姓 | `Jean-Pierre Bardet` | `J.~P. Bardet` |
| 特殊字符 | `Fran{\c{c}}ois Fleuret` | `Francois Fleuret` |
| 多作者 et al. | `et~al.` | `et al.` |
| 波浪号连接 | `Aidan~N Gomez` | `Aidan N Gomez` |

### 3.2 中文作者

| 规则 | 格式 | 示例 |
|------|------|------|
| 拼音 | 姓, 名 | `Li, Hengbo` |
| 多字名 | 姓, 名首字母 | 无 |

## 四、引用分类规则 / Citation Classification Rules

### 4.1 必需引用（每篇论文必须引用）

| 论文类型 | 必需引用 | 说明 |
|----------|---------|------|
| 所有论文 | `vaswani2017attention` | Transformer 基线 |
| Theory 篇 | `lsh-sfa` | LSH-SFA 核心架构（含时空引擎） |
| Architecture 篇 | `lsh-sfa`, `lsh-three-layer` | 架构基础 |
| Systems 篇 | `lsh-sfa`, `encoding-gradient` | 实现基础 |
| Applications 篇 | `lsh-sfa` | 应用基础 |
| Ethics 篇 | `lsh-sfa`, `lsh-three-layer` | 伦理基础 |

### 4.2 领域必需引用

| 论文主题 | 必需引用 | 说明 |
|----------|---------|------|
| PDE/物理 | `raissi2019physics`, `chen2018neural` | PINN, Neural ODE |
| 高斯相似度 | `rasmussen2006gaussian`, `broomhead1988multivariable` | GP, RBF |
| 线性注意力 | `katharopoulos2020transformers`, `choromanski2021rethinking` | Linear Attn, Performer |
| ELU+1 | `clevert2016fast`, `tsai2019transformer` | ELU, Kernel View |
| 自回归 | `radford2019language`, `peng2023rwkv` | GPT, RWKV |
| 分词器 | `sennrich2016neural` | BPE |
| Rust/系统 | `burn2024`, `pyo32024` | 框架引用 |

### 4.3 禁止引用

| 禁止 | 原因 |
|------|------|
| arXiv preprint ID 作为标题 | 应使用正式发表版本 |
| 维基百科 | 非学术来源 |
| 未验证的博客文章 | 非同行评审 |

## 五、引用完整性约束 / Citation Completeness Constraints

### 5.1 \cite 与 \bibitem 一致性

| 规则 | 说明 |
|------|------|
| 每个 `\cite{key}` 必须有对应 `\bibitem{key}` | 编译不报错 |
| 每个 `\bibitem{key}` 必须被 `\cite{key}` 引用 | 无孤立引用 |
| 同一引用键在所有论文中格式一致 | 跨论文一致性 |

### 5.2 年份准确性

| 常见错误 | 正确年份 |
|----------|---------|
| QLoRA (2024) | 2023 (NeurIPS 2023) |
| Clark "Being There" (1998) | 1997 (MIT Press 1997) |
| Floridi "Philosophy of Information" (2010) | 2011 (OUP 2011) |

### 5.3 标题大小写

| 规则 | 正确 | 错误 |
|------|------|------|
| 句首大写 | `Transformers are RNNs` | `Transformers are rnns` |
| 专有名词保留 | `Attention is all you need` | `Attention Is All You Need` |
| 缩写保留 | `Fast autoregressive transformers` | `Fast Autoregressive Transformers` |

## 六、检查清单 / Checklist

### 6.1 提交前检查

- [ ] 所有 `\cite{key}` 有对应 `\bibitem{key}`
- [ ] 所有 `\bibitem{key}` 被至少一个 `\cite{key}` 引用
- [ ] 外部引用年份准确（对照原论文）
- [ ] 作者姓名完整（非缩写）
- [ ] 标题大小写正确（句首大写+专有名词）
- [ ] 会议/期刊名称正确（NeurIPS 不是 NIPS）
- [ ] 内部引用键与统一表一致
- [ ] 领域必需引用已包含
- [ ] 无 arXiv ID 作为标题
- [ ] 无孤立引用（未被 \cite 引用的 \bibitem）

### 6.2 年度审计

- [ ] 检查所有外部引用是否有更新版本（如 arXiv → 正式发表）
- [ ] 检查 DOI 是否有效
- [ ] 检查新发表的相关论文是否需要补充
